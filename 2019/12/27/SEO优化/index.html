SEO优化(又叫搜索引擎优化)

1.为每一个页面添加关键字和描述信息（添加在 header 标签中）

2.为每一个 img 设置一个 alt 属性

3.优化网页加载速度

4.不要在页面中显示空连接

5.写代码要规范，使用语义化标签，有助于网页优化

6.在网页中尽量设置 h1-h6 标签，增加 SEO 的命中权重

7.定期更新网站内容，发布新文章，增加搜索引擎的收录

vue 中如果做 seo 优化

设置一个单独的用于 seo 搜索的网站

在蜘蛛爬取网站的时候把搜索引擎的抓取过程转接到另一个单独的网站。这个网站主要是一些静态页面，是传统的多页面网站，主要用来做 seo 优化的。可以没有 css 样式，没有 js 代码，只有 html 标签。

nuxt

nuxt 是一个 vue 的框架，可以实现服务器端渲染。可以使用 vue 创建多页面应用程序。
